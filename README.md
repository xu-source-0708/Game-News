# GameNews_FP UX - 游戏新闻聚合看板

这是一个基于HTML、CSS和JavaScript的游戏新闻聚合看板，可以集成多个RSS源并以网格布局展示最新新闻。

## 功能特点

1. **橙色主题设计** - 采用类似FunPlus的橙色主题，视觉效果醒目
2. **自适应网格布局** - 根据屏幕宽度自动调整每行显示的新闻板块数量：
   - 手机：每行1-2个板块
   - 平板：每行2-3个板块
   - 桌面：每行3-4个或更多板块
3. **新闻标题两行显示** - 确保新闻标题在两行内完整显示
4. **预设RSS源** - 包含6个游戏相关的预设RSS源
5. **自定义RSS源** - 支持添加和删除自定义RSS源
6. **本地存储** - 保存用户添加的RSS源到浏览器本地存储
7. **响应式设计** - 适配不同屏幕尺寸的设备

## 预设RSS源

- Mobile Gamer
- 游戏矩阵
- GamesIndustry.biz
- GameDeveloper

## 使用方法

1. 打开 `index.html` 文件即可使用
2. 点击右上角的"添加RSS源"按钮可以添加自定义RSS源
3. 自定义RSS源可以通过点击右上角的×按钮删除
4. 点击新闻标题可以打开链接（注意：由于浏览器跨域限制，当前版本使用的是模拟数据）

## 开发说明

- 由于浏览器的跨域限制，当前版本使用模拟数据展示RSS内容
- 在实际部署时，可以考虑添加后端代理来解决跨域问题
- 支持键盘快捷键：Ctrl+R 刷新所有RSS源，Esc 关闭模态框

## 技术栈

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6+)
- 本地存储 (localStorage)
